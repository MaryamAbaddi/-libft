# Overview
Libft is a custom C library developed as part of the 42 curriculum.
The goal of this project is to recreate essential functions from the standard C library and implement additional utility functions to deepen understanding of low-level programming in C.

This project focuses on:
- Memory management
- String manipulation
- Data structures (linked lists)
- Understanding pointers and algorithms
- Writing clean and efficient C code

# Objectives
- Reimplement standard C library functions from scratch.
- Improve understanding of how built-in functions work internally.
- Strengthen problem-solving and debugging skills in C.
- Build a reusable library for future projects.
